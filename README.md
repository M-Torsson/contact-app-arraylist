# Contact Management App

This is a simple Java console application for managing contacts.

The program was created as part of a workshop to practice Java Collections.

## Features

- Add a contact (name and mobile number)
- Search for a contact by name
- Display all contacts
- Exit the program

## How it works

The program stores contacts using an **ArrayList**.  
Each contact is stored as a string in this format:

name|mobile

Example:

Ali | 0700000000

## Technologies

- Java
- ArrayList (Java Collections)
- Scanner for user input
